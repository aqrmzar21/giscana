## 🚀 Cara Menggunakan:

### **Untuk Pengunjung Umum:**
1. Buka `http://localhost:8000` untuk melihat landing page
2. Scroll untuk melihat fitur-fitur sistem
3. Klik "Daftar Sekarang" untuk membuat akun
4. Atau klik "Lihat Peta Interaktif" untuk akses langsung

### **Untuk User yang Sudah Login:**
1. Login dengan akun yang sudah dibuat
2. Dashboard akan menampilkan welcome message
3. Klik "Akses Peta Interaktif" untuk masuk ke sistem GIS
4. Gunakan navigasi untuk berpindah antar halaman

### **Akun Default untuk Testing:**
- **Admin**: admin@giscana.local / password
- **Staff**: staff@giscana.local / password
